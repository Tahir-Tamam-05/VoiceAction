# VoiceAction App Analysis

## 1. What is This App?

**VoiceAction** is a voice-first note-taking application that uses speech recognition and AI (Google Gemini) to transform voice memos into structured, organized notes.

## 2. Core Features

### Voice Recording
- Real-time speech-to-text using Web Speech API
- Live transcription display with waveform visualization
- Timer to track recording duration
- Auto-save on page unload

### AI Processing
- Uses Google Gemini API to process voice notes
- Automatically extracts titles, content, type (task/event/idea), and mood
- Falls back gracefully if AI processing fails

### Note Management
- Create, edit, delete notes
- Pin important notes
- Tags support (e.g., BUSINESS, PERSONAL, IDEAS)
- Note types: task, event, idea, audio, voice, text

### Additional Features
- Streak tracking (consecutive days of notes)
- Dark/Light mode toggle
- Quick capture (type + AI assist)
- Flashcards generation
- Translation support
- Link detection/extraction

### User Authentication
- Firebase authentication
- User profiles with name, email, avatar

## 3. Architecture

### Tech Stack
- **Frontend**: React 19 + Vite + TypeScript
- **Styling**: Tailwind CSS 4 + Motion (Framer Motion)
- **3D**: React Three Fiber (for visual effects)
- **AI**: Google Gemini API
- **Auth**: Firebase
- **Icons**: Lucide React

### Key Files
- `src/App.tsx` - Main app router and state management
- `src/Home.tsx` - Home screen with quick capture, pinned/recent notes
- `src/Recording.tsx` - Voice recording screen
- `src/hooks/useSpeechRecognition.ts` - Web Speech API wrapper
- `src/hooks/useNotes.ts` - Note CRUD operations
- `src/services/geminiService.ts` - AI processing

### Screens
1. **Landing** - Pre-auth welcome screen
2. **SignIn** - Firebase authentication
3. **Home** - Main dashboard
4. **Search** - Search notes
5. **History** - All notes view
6. **Settings** - User preferences
7. **Recording** - Voice capture
8. **EditNote** - Edit individual note
9. **Flashcards** - Flashcard view

## 4. How It Works

1. User authenticates via Firebase (or skips for quick demo)
2. User can:
   - Tap voice button to start recording
   - Type a quick note with AI enhancement
3. When recording stops, audio is processed through Gemini API
4. AI extracts structured data (title, type, mood, body)
5. Note is saved to Firebase (per user)
6. User can view, edit, pin, tag, or delete notes
7. Streak tracks usage consistency

## 5. Current Status

**Status**: Functional MVP

- Voice recording works (Web Speech API)
- AI processing works (Gemini API)
- Notes persist to Firebase
- Basic UI with animations
- Dark mode support
- Flashcards feature in progress

## 6. Potential Issues / Notes

- Web Speech API requires Chrome/Edge (not supported in Safari/Firefox)
- Needs `GEMINI_API_KEY` in `.env.local`
- Firebase config needs to be configured
- Some features like translation and flashcards are partially implemented
